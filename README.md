User Management REST API

A simple REST API built using Node.js and Express.js.

Features

* Get all users
* Get a user by ID
* Create a new user
* Update an existing user
* Delete a user

Technologies Used

* Node.js
* Express.js
* File System (fs)
* Postman
* Git & GitHub

Installation

1. Clone the repository
2. Install dependencies

npm install

3. Start the server

npm start

API Endpoints

Get all users

GET /api/users

Get user by ID

GET /api/users/:userID

Create user

POST /api/users

Update user

PATCH /api/users/:userID

Delete user

DELETE /api/users/:userID
